Quick Sort is a widely used sorting algorithm based on the divide-and-conquer strategy. It selects a pivot element, partitions the list into two sublists, and recursively sorts them. It has an average time complexity of O(n log n), making it efficient for large datasets.

Sample Quick Sort Algorithm

    def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
