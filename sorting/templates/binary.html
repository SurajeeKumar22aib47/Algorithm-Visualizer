Binary Search Visualizer

Binary Search is an efficient searching algorithm for sorted lists. It repeatedly divides the list in half and compares the target value with the middle element. If the target is smaller, it searches the left half; if larger, the right half. It continues this process until the target is found or the search space is empty. With a time complexity of O(log n), it is suitable for large datasets.

    def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
